# NPM IntelliSense

NPM IntelliSense est une extension VS Code qui autocomplète les noms des modules npm dans les déclarations d'import.

## Installation

1. Ouvrez VS Code
2. Appuyez sur `Ctrl+P` / `Cmd+P`
3. Collez `ext install christian-kohler.npm-intellisense`
4. Appuyez sur Entrée

## Fonctionnalités

- Autocomplétion des noms de modules npm lors de l'import
- Suggestions basées sur les dépendances dans package.json
- Fonctionne avec les imports CommonJS et ES6
- Support des chemins relatifs

<Callout type="info">
L'extension analyse votre package.json pour suggérer uniquement les modules que vous avez installés.
</Callout>

## Utilisation

Commencez à taper une déclaration d'import et les suggestions apparaîtront automatiquement :

``` js
import { } from 'ex' // Suggère express, expo, etc.
const module = require('re') // Suggère react, redux, etc. 
```

## Configuration

Quelques options sont disponibles dans les paramètres VS Code :

- `npm-intellisense.scanDevDependencies`: Inclure les devDependencies dans les suggestions
- `npm-intellisense.recursivePackageJsonLookup`: Rechercher récursivement package.json
- `npm-intellisense.packageSubfoldersIntellisense`: Activer l'autocomplétion pour les sous-dossiers

## Dépannage

Si l'autocomplétion ne fonctionne pas :

1. Vérifiez que l'extension est bien installée et activée
2. Assurez-vous d'avoir un fichier package.json valide
3. Redémarrez VS Code
