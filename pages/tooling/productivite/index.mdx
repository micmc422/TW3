
# **Productivité Générale**  

## **Prettier : Formatage automatique du code**  
Prettier est un formateur de code qui applique un style cohérent à vos fichiers. Il automatise le formatage selon des règles préconfigurées, éliminant ainsi les discussions inutiles sur le style de code dans les équipes.  

**Pourquoi l'utiliser ?**  
- Gain de temps grâce à un formatage instantané.  
- Respect des conventions définies dans le projet.  

## **ESLint : Contrôle qualité du code**  
ESLint est un outil d’analyse statique qui détecte les erreurs et garantit que le code suit les bonnes pratiques.  

**Pourquoi l'utiliser ?**  
- Identifie les erreurs potentielles dès l’écriture.  
- Permet de personnaliser les règles selon les besoins du projet.  

## **Path Intellisense : Autocomplétion de chemins**  
Un plugin pratique pour les éditeurs comme VSCode, qui suggère et complète automatiquement les chemins de fichiers lorsque vous les importez.  

**Avantages :**  
- Réduction des erreurs de chemin.  
- Productivité accrue lors de l'importation de modules.  

## **Bracket Pair Colorizer : Coloration des accolades**  
Cet outil colore les paires d’accolades, crochets ou parenthèses, facilitant ainsi leur identification dans le code complexe.  

**Avantages :**  
- Amélioration de la lisibilité.  
- Facilité pour repérer les erreurs de fermeture d'accolades.  

## **Material Icon Theme : Icônes de fichiers**  
Une extension visuelle pour assigner des icônes distinctives aux différents types de fichiers dans votre projet.  

**Avantages :**  
- Meilleure organisation visuelle.  
- Navigation plus rapide dans un projet complexe.
