# 🚀 **Installation de Docker sur macOS**

*Les postes de l'université sont déjà configurés avec Docker et Ubuntu.*

Les prérequis et la procédure d'installation de Docker varient en fonction du système d'exploitation. Voici comment installer **Docker Desktop** sur macOS :

---

## 🖥️ **Prérequis pour macOS**

Avant d'installer Docker sur macOS, assurez-vous que votre système respecte ces prérequis :

- **Système d'exploitation :**  
  Docker Desktop pour macOS nécessite **macOS Yosemite** ou une version supérieure, avec la virtualisation matérielle activée. 🔧
  
- **Processeur :**  
  Votre Mac doit disposer d'un processeur **Intel** prenant en charge la virtualisation matérielle. ⚙️

- **Mémoire :**  
  Il est recommandé d'avoir **au moins 4 Go de RAM** pour garantir des performances optimales. 💻

- **Espace disque :**  
  Docker nécessite **20 Go d'espace disque libre** pour stocker les images et conteneurs Docker. 📦

- **Compatibilité hôte :**  
  Vérifiez que votre version de macOS est prise en charge, car Docker Desktop s'appuie sur la virtualisation intégrée de macOS. 📱

> ℹ️ **Note importante :**  
> Pour les prérequis les plus récents et détaillés, consultez la [documentation officielle de Docker](https://docs.docker.com/engine/install/).

---

## ⚙️ **Installation de Docker Desktop sur macOS**

Suivez ces étapes pour installer Docker Desktop sur votre Mac :

1. **Téléchargement :**  
   Allez sur le site officiel de Docker et téléchargez **Docker Desktop pour macOS** :  
   ➡️ [Télécharger Docker Desktop](https://www.docker.com/products/docker-desktop)

2. **Installation :**  
   - Double-cliquez sur le fichier téléchargé pour monter l'image disque. 📥
   - Glissez l'icône **Docker** dans le dossier **Applications**. 🗂️

3. **Lancement :**  
   - Ouvrez Docker depuis le dossier **Applications**. Une fois lancé, Docker se mettra en arrière-plan pour gérer vos conteneurs. 🚀

> 📝 **Astuce :**  
> Docker Desktop se lance automatiquement à chaque démarrage de votre Mac et fonctionne en arrière-plan.

---

## 🔍 **Vérification de l'installation**

Après l'installation, vérifiez que Docker fonctionne correctement en exécutant ces commandes dans le terminal :

```bash copy
docker --version
docker -v  # Version courte
```

Ces commandes vous indiqueront la version de Docker installée sur votre Mac. Vous devriez voir quelque chose comme :

```bash copy
Docker version 20.10.7, build f0df350
```
