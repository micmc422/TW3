# 🚀 Installation de Docker

**Les postes de l'université sont déjà configurés avec Docker et Ubuntu.**  
Si vous devez installer Docker sur un autre poste, voici la procédure à suivre.

---

## 🖥️ Prérequis pour Linux (Ubuntu)

- **Système d'exploitation :** Ubuntu (version compatible avec Docker).
- **Droits d'administration :** Vous devez disposer des droits `sudo` sur votre système.
- **Espace disque :** Minimum 10 Go d'espace libre pour les images et les conteneurs Docker.
- **Architecture :** Votre système doit être compatible avec Docker (par exemple, architecture **x86_64/amd64**).

**Important :** Vérifiez régulièrement la documentation officielle de Docker pour obtenir les informations les plus récentes sur les prérequis en fonction de la version installée. [Docker Documentation](https://docs.docker.com/engine/install/)

---

## ⚙️ Installation de Docker

### Étapes à suivre :

1. **Mettez à jour les paquets de votre système :**
    ```bash
    sudo apt-get update
    sudo apt-get upgrade
    ```

2. **Installez Docker :**
    ```bash
    sudo apt install docker.io
    ```

3. **Démarrez Docker et activez-le au démarrage :**
    ```bash
    sudo systemctl start docker
    sudo systemctl enable docker
    ```

Docker est désormais installé et prêt à être utilisé sur votre machine Ubuntu ! 🐳
