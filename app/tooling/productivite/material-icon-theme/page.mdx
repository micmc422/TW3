# 🖼️ Material Icon Theme

Material Icon Theme est une extension pour Visual Studio Code qui ajoute des icônes Material Design à vos fichiers et dossiers dans l'explorateur de fichiers.

## 🛠️ Installation

1. Ouvrez VS Code
2. Allez dans l'onglet Extensions (Ctrl+Shift+X)
3. Recherchez "Material Icon Theme"
4. Cliquez sur "Install"

## 📘 Fonctionnalités

- Plus de 150 icônes Material Design
- Prise en charge de nombreux langages et frameworks
- Icônes personnalisées pour les dossiers
- Thème sombre et clair
- Personnalisation des icônes

## 🛠️ Configuration

Vous pouvez personnaliser les icônes dans les paramètres VS Code :

1. Ouvrez les paramètres (Ctrl+,)
2. Recherchez "material-icon"
3. Modifiez les options selon vos préférences

## 📘 Avantages

- Meilleure organisation visuelle
- Navigation plus intuitive
- Identification rapide des types de fichiers
- Aspect professionnel de l'éditeur

Material Icon Theme est un excellent moyen d'améliorer l'expérience visuelle de VS Code tout en facilitant l'identification des fichiers dans vos projets.
