# Node.js Spécifiques

Les extensions suivantes sont spécifiquement conçues pour améliorer le développement Node.js :

- **npm** : Intégration complète du gestionnaire de paquets npm directement dans VS Code. Permet d'exécuter des scripts npm, de gérer les dépendances et d'interagir avec le registre npm.

- **npm Intellisense** : Fournit l'autocomplétion intelligente pour les modules npm dans les déclarations d'importation. Suggère automatiquement les noms des paquets disponibles lors de l'écriture des instructions require ou import.

- **Node.js Modules Intellisense** : Extension qui améliore l'autocomplétion pour les modules Node.js natifs et installés. Facilite l'importation des modules en suggérant les chemins et les exports disponibles.

- **REST Client** : Permet de tester les API REST directement depuis VS Code. Supporte l'envoi de requêtes HTTP, la visualisation des réponses et la gestion des environnements de test.

- **Docker** : Offre une intégration complète de Docker pour le développement Node.js. Permet de gérer les conteneurs, de construire des images et de déboguer les applications Node.js conteneurisées.
