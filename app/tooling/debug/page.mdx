# Débogage & Performance

Le débogage est une étape cruciale dans le développement d'applications React. Cette section explore différents outils et techniques pour faciliter le débogage et optimiser les performances de vos applications :

- **Débogage avec Chrome** : Utilisation des Chrome DevTools pour inspecter et déboguer votre application React en temps réel
- **VSCode débogage** : Configuration et utilisation du débogueur intégré de VSCode pour React
- **Performance Monitor** : Surveillance et analyse des performances de votre application
- **Turbo Console Log** : Extension VSCode pour accélérer l'ajout de logs de débogage

Ces outils vous permettront d'identifier et résoudre rapidement les problèmes dans votre code, tout en maintenant des performances optimales.
