## 5. Git & Collaboration

Git est un système de contrôle de version distribué essentiel pour le développement collaboratif. Dans ce chapitre, nous allons explorer trois extensions VS Code qui améliorent considérablement l'expérience de travail avec Git :

- **GitLens** : Une extension puissante qui enrichit les fonctionnalités Git intégrées de VS Code, permettant de visualiser l'historique des modifications, les auteurs du code, et bien plus encore.

- **Git History** : Un outil qui offre une visualisation claire et interactive de l'historique des commits, des branches et des modifications.

- **GitHub Pull Requests** : Une extension qui permet de gérer les pull requests directement depuis VS Code, facilitant la revue de code et la collaboration.

Ces outils combinés permettent d'optimiser votre workflow Git et d'améliorer la collaboration au sein de votre équipe de développement.
